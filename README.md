# Summary

R package for reading US mortality data from the [National Center for Health Statistics](https://www.cdc.gov/nchs/index.htm) (NCHS).

# Installation

To install the package, use `install_github()` from the *remotes* package or `pkg_install()` from the *pak* package:

```r
pak::pkg_install('yea-hung/readUSmortality')
```
